## 为什么1KB等于1024B

### 如下是用二进制表示1000与1024
```
01				--2
001				--4
0001 			--8
00001			--16
000001			--32  
0000001			--64
00000001		--128
000000001		--256
0000000001		--512
0001011111		--1000
00000000001		--1024

256 512 128 64	32	8
    768 896 960 992 1000
```

### 然后再用进行单位转换：
假设：1KB = 1000B 即有： 1KB = 1000 * 2^3 bit

而 1KB = 1024B 时，1KB = 2^10 * 2^3 bit = 2^13 bit


### 结论
1. 1KB = 1024B, 方便单位运算
2. 1KB = 1024B, 1024在二进制中最接近1000
