### 1、显示当前所在目录
```cmd
cd
```

### 2、盘符切换

eg:从C盘切换到D盘
```cmd
D:
```

### 3、查看当前目录文件
```cmd
dir
```

### 4、当前盘符内目录切换
```cmd
cd + path
```
