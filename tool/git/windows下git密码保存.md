### windows 下git密码保存

#### 方法一
在win10或者win7都是一样的步骤：
```
（一）进入控制面板

（二）选择用户账户

（三）选择管理你的凭据

（四）选择Windows凭据

（五）选择git保存的用户信息

（六）选择编辑或者进行删除操作

（七）完成
```
